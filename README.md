\# DevOps Learning Project



This project demonstrates basic Docker, Linux, and Git operations.



\## Five DevOps Concepts:



1\. \*\*Infrastructure as Code\*\* - Defining infrastructure in configuration files

2\. \*\*Containerization\*\* - Packaging applications in isolated environments  

3\. \*\*Version Control\*\* - Tracking changes with Git

4\. \*\*CI/CD\*\* - Automated testing and deployment

5\. \*\*Monitoring\*\* - Tracking system performance



\## Project Contents:

\- Dockerfile with Linux tools

\- Basic Linux command examples

\- Git version control setup



\## How to Use:

1\. Build the image: `docker build -t linux-demo .`

2\. Run the container: `docker run -it linux-demo`

3\. Explore Linux commands inside the container
